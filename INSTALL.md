# Install Ask Huberman

## Prerequisites
- Claude Code (CLI or Desktop)

## Installation

1. Clone this repo:
   ```bash
   git clone https://github.com/cdeistopened/ask-huberman.git
   cd ask-huberman
   ```

2. Open in Claude Code:
   ```bash
   claude
   ```
   Or open the folder in Claude Code Desktop.

3. Start with:
   > "I want to optimize my health"

   The plugin will run an intake assessment and route you to the right protocols.

## What's Inside
- 6 interactive skills (assessment → foundation → protocols → check-in → deep dive)
- 12 framework articles with specific dosages, timing, and mechanisms
- 137 searchable podcast transcripts (full progressive disclosure)
- A/B test protocol for comparing against vanilla ChatGPT

## Skills
| Skill | Purpose |
|-------|---------|
| `ask-huberman` | Concierge router — figures out what you need |
| `full-intake` | Health assessment — builds your profile |
| `foundation-fix` | Identifies and fixes your weakest pillar |
| `protocol-builder` | Designs your full weekly protocol |
| `check-in` | 2-week reassessment and adjustment |
| `deep-dive` | Teaches the mechanism behind any protocol |
